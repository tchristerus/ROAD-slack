# about
ROAD server. Used as link between github / slack and the duck.

# Installation
`npm install`

# iptables
`ufw allow 6666`

`ufw allow 1234` 
## run
`node App.js`


## github integration
add webhook (json) to your repo (https://slack.christerus.com/github/end)

## slack integration
Add ROAD plugin to your workgroup
